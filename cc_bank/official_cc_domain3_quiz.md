## Question 1

**Which of the following is an example of a security control?**

- Firewall
- Web browser
- Computer mouse
- Network cable

**Answer:** Firewall

**Explanation:** A firewall is a technical security control designed to restrict network traffic.

---

## Question 2

**What is the definition of an object in the context of access controls?**

- An entity that responds to a request for service
- A device with onboard firmware
- Something that contains its own access control logic
- Anything that initiates a request for service

**Answer:** An entity that responds to a request for service

**Explanation:** Objects are passive resources that subjects attempt to access.

---

## Question 3

**Derrick logs on to a system to read a file. In this example, Derrick is the ______.**

- Predicate
- Process
- Object
- Subject

**Answer:** Subject

**Explanation:** A subject is an active entity that initiates access.

---

## Question 4

**Which of the following is a subject?**

- Filename
- Fence
- File
- User

**Answer:** User

**Explanation:** A user is an active entity requesting access.

---

## Question 5

**What strategy integrates people, technology, and operations to create multiple layers of protection?**

- Cyberattack Prevention
- Layered Defense
- Multi-Factor Authentication
- Single Point of Failure

**Answer:** Layered Defense

**Explanation:** Layered defense (defense in depth) uses multiple controls across layers.

---

## Question 6

**How does privileged access management implement least privilege?**

- Restricting access to critical information only
- Granting maximum access to all users
- Providing access based on seniority
- Granting users only what they need

**Answer:** Granting users only what they need

**Explanation:** Least privilege limits access strictly to job requirements.

---

## Question 7

**In Mandatory Access Control (MAC), what determines access levels?**

- Government policy and security clearance
- Owner discretion
- Individual judgment
- Request basis

**Answer:** Government policy and security clearance

**Explanation:** MAC is centrally enforced and not user-controlled.

## Question 10

## Question 12
